V-Shift

V-Shiftアプリケーションです。インストーラーをダウンロードして、お使いのWindowsコンピューターにインストールしてください。

ダウンロード

バージョン 1.1 • Windows 10/11 対応 • 無料

エクセルテンプレート

V-Shiftにインポート可能なエクセルシフト表の入力形式テンプレートです。既存のエクセルファイルがある場合に便利です。

ダウンロード

Excel 2016以降対応 • .xlsx形式

インストール手順

  1. https://v-shift-dev.github.io/V-Shift/にアクセスし、"V-Shiftをダウンロード"をクリック
  2. ダウンロードしたzipファイルを解凍
  3. 解凍したフォルダを"C:\Program Files"に移動
  4. フォルダ内の"V-Shift.exe"のショートカットをデスクトップ等任意の場所に配置
  5. ショートカットから"V-Shift.exe"を実行
  6. 初回起動時にインストールも自動で行われます。完了したら自動的に起動します。
注意事項：
  • 起動時は、必ず"V-Shift.exe"から実行してください。
  • 初回起動時、Windowsのセキュリティ警告が表示される場合がありますが、問題ありません。"詳細情報"をクリックし、"実行"を選択してください。
  • "C:\Program Files"内に配置したV-Shiftフォルダ内に、アプリ実行に必要なファイル群が格納されています。これらのファイルは移動、削除を行わないで下さい。

アンインストール手順

  1. V-Shiftを終了
  2. "C:\Program Files\V-Shift"フォルダを削除
  3. デスクトップ等に配置したショートカットを削除

機能

シフト管理業務を効率化する豊富な機能

直感的なシフト作成

スタッフの休み予定を考慮した効率的なシフト作成。看護師・介護士のシフト表を簡単に作成できます。

自動時間計算

夜勤・日勤・休日出勤など、様々な勤務パターンの合計時間を自動で計算します。

Excel連携

既存のExcelシフト表からデータを取り込み、Excelへの出力も対応。既存業務との統合をスムーズに。

スタッフ情報管理

介護職員・看護職員・ヘルパーの基本情報を一元管理。休み予定の登録も簡単です。

ダブルブッキング防止

自動チェック機能でダブルブッキングを防止。シフト作成時のミスを大幅に削減します。

小・中規模施設最適化

小・中規模の介護施設に最適化された機能設計。シンプルで使いやすいインターフェースを提供します。

月表示モードでシフト全体を把握

月単位でのシフト表示により、全体のスケジュールを一目で確認。カレンダー形式のレイアウトでスタッフの勤務状況や休暇予定を効率的に管理。色分け表示により職員の判別も簡単。

職員と利用者の表示切り替え

職員表示と利用者表示をワンクリックで切り替え可能。職員のシフトをベースに、利用者ごとのケア予定も同じ画面で一元表示。介護現場に必要な情報を直感的に確認でき、業務の見通しと効率を大幅に向上。

ダブルブッキング自動チェック

シフト編集時の自動チェックに加え、全体チェック機能も搭載。ワンクリックですべてのシフトに対して重複がないかを一括確認可能。ミスを事前に防ぎ、トラブルのない現場づくりを。
